# Kindreds of Middle-earth — Древа умений всех рас (подробно)

_Автоматически сгенерировано из данных мода. Для каждого узла указаны: уровень (Ур.), стоимость в очках дисциплины (оч.), пометка **эксклюзив** (в группе можно взять только одну такую вершину), и эффект._

**Как читать эффекты:**

- `[пассив/атрибут]` — постоянная прибавка к характеристике.
- `[пассив]` — постоянная способность.
- `[аура]` — действует на вас и союзников рядом.
- `[аура/визуал]` / `[пассив/визуал]` — есть видимый эффект (свечение врагов, частицы и т.п.).
- `[КОНТЕКСТ]` — работает только в определённой обстановке (во тьме, под землёй, при низком здоровье и т.д.).
- `[АКТИВ]` — активная способность, ставится в слот и применяется вручную; указана перезарядка.
- `[стрельба]` — влияет на стрелы из лука/пращи.

> Примечание: тёмные расы (Орк, Снага, Гоблин) слабеют при свете дня (штраф к урону и Замедление — врождённая черта), но сильнее во тьме. Урук-хай — исключение: он **силён на солнце**.
