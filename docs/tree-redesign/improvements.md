## The core principle we're executing: the "verb test"
A meaningful node has a **verb** (pierce, curve, ignite, revive, summon); a *great* one has a **unique**
verb. Pure "+X stat" nodes are filler. Benchmark: AC Origins ≈ 76% meaningful nodes; weak trees ≈ 17%.
→ **The per-discipline redesign (Elf Archery done) is the fix. Roll it across every discipline of every
race. No filler.**

## Systemic upgrades (do alongside the per-discipline passes)
1. **Force build identity** — exclusive keystones + a soft cap (master 2–3 disciplines, dabble in rest).
   Right now everything is maxable → no identity.
2. **Costed respec** — allow full respec but at a real price (XP / an item), so choices commit.
3. **Capstones = unique verbs**, not stat combos — each a genuine game-changer.
4. **Feedback cadence** — reward ~every 30–90s, milestone ~every 10–15 min; early nodes must be *felt*
   (we're doing this), add discipline level-up milestones.
5. **Paths + titles** (from PMMO Classes) — frame each race's branches as named Paths (Elf: Warden /
   Loremaster / Minstrel) with a signature passive + title. More identity, cheap to add.

## Signature feature (unique, no other mod has it)
- **Fellowship / synergy** — Free Peoples grow stronger among **diverse allies**; orc-kin from
  **numbers**. Designed earlier, never built. This is the heart of LOTR — our standout.

## Deep LOTR lore to tap (real, fun, per race)
- **Elf:** foresight (danger-sense pre-buff), song-magic (crowd-control/enchant), star-light wards, the
  Eldar don't truly die (a lingering "fade" revive).
- **Dwarf:** master-craft (better/unbreakable gear, mithril-working, repair), stone-doors,
  corruption-resistance (have).
- **Men:** Gift of Men (defiance/second-wind), ambition (faster learning), kingship auras.
- **Hobbit:** unexpected bravery (clutch power), thrown stones, good-cheer food auras.
- **Race-craft (big content pass):** lembas (great food), elven-cloak (camouflage), mithril, Dwarf-armor
  — real craftable items per race, lore-first. No mod does this.
