# Kindreds — Руководство по тестированию (шаг за шагом)

## 0. Подготовка (делать каждый раз перед тестом)
1. **Полностью закрой Minecraft**, затем запусти заново (jar нельзя подменять на лету).
2. Создай тестовый мир с **читами ВКЛ** (нужны для админ-команд `/kindreds`).
   - В существующем мире: пауза → **Открыть для сети** → *Разрешить читы: ВКЛ* → Запустить.
3. Выбери расу в базовом моде (**Книга игрока** при первом входе). Чтобы протестить другую расу — снова через Книгу игрока / смену расы базового мода.
4. Полезные клавиши (переназначаются в Настройки → Управление → «Kindreds»): **K** = древо навыков, **J** = Кодекс, **V** = смена зрения, **G** = применить активную способность.

**Основные команды**
| Команда | Что делает |
|---|---|
| `/kindreds inspect` | Показывает расу, уровни/очки дисциплин, врождённые плюсы и минусы |
| `/kindreds grantxp <дисциплина> <кол-во>` | Быстро набить очки (напр. `/kindreds grantxp combat 8000`) |
| `/kindreds respec` | Сбросить все открытые узлы |
| `/kindreds codex` | Получить книгу-Кодекс |
| `/kindreds config` | Показать все значения конфига |
| `/kindreds config <ключ> <значение>` | Задать одно (напр. `/kindreds config enableBirthTraits false`) |

Дисциплины для grantxp: `combat, archery, mining, stealth, smithing, survival, lore, song, beast_lore, runecraft, leadership, shadow`.

## 1. Кодекс и интерфейс (любая раса)
- [ ] В инвентаре **сразу есть книга Kindred Codex**. **ПКМ по ней** → открывается Кодекс-пергамент. Листай **◄ Prev / Next ►** по всем 8 народам. *Ожидается:* книжный вид, плюсы зелёным ✦, минусы красным ✖, вверху — подсказки.
- [ ] Нажми **J** в мире → тот же Кодекс.
- [ ] Начни НОВОГО персонажа → на экране выбора расы базового мода вверху справа кнопка **«⚔ Kindred Traits»** → открывает Кодекс.
- [ ] Нажми **K** → древо открывается в режиме **Карта**: все дисциплины как цветные области. **Тяни** мышью, **колёсиком** приближай/отдаляй. Кнопка **◇ Whole map / ▤ One branch** (справа вверху) — переключение вида. Слева показаны **только дисциплины твоей расы**. *Ожидается:* читаемо, без фиолетовых/битых текстур, без обрезки.

## 2. Врождённые черты по расам
Для каждой расы сначала `/kindreds inspect` (там список плюсов/минусов), затем проверь в мире. Эффекты-зелья показываются **иконками** в инвентаре/интерфейсе.

### 🌟 Эльф
- [ ] +1 сердце (всего 11), постоянное **Ночное зрение**.
- [ ] Атаки заметно **быстрее** (+15% скорости атаки); мобы замечают **позже** (скрытность).
- [ ] **Ночью под открытым небом** → иконки **Регенерация + Скорость** (Благодать звёзд). Под крышу/под землю или днём → исчезают.
- [ ] Не голодает; ничего не ешь — шкала голода стоит. Эффекты **Тошнота/Голод** не держатся; в снежном порошке **не замерзает**.
- [ ] В **тёмной неосвещённой пещере без неба** → **Слабость** (неуют в глубокой тьме). Освети/выйди → пропадает.
- [ ] Получи вредный эффект (напр. брызни зельем Яда) → он **проходит в ~3 раза быстрее**.

### 🔥 Гном
- [ ] +3 сердца (всего 13). Встань в **огонь/лаву** → урон есть, но **горишь заметно меньше** (не иммунитет).
- [ ] **Под землёй (нет неба сверху)** → иконки **Спешка + Сопротивление**. На поверхности → пропадают.
- [ ] Копаешь камень заметно быстрее. Передвижение **на ~10% медленнее**, чем у человека.
- [ ] Яд/проклятия проходят в ~3 раза быстрее; голод убывает медленнее.

### 🍃 Хоббит
- [ ] Мобы замечают **очень поздно** (подкрадись — реагирует медленно). **+3 удача**.
- [ ] Пока **еда 18+**, медленно **регенерирует** здоровье.
- [ ] −1 сердце (всего 9), слаб в ближнем бою (−20%). Вредные эффекты проходят быстро (стойкость к Кольцу).

### ⚔️ Человек
- [ ] +2 сердца, немного сильнее удар, чуть больше **дальность** (бьёшь/ставишь чуть дальше).
- [ ] **Забеги на лёд** → **НЕ скользишь** (Странник). Другой расой — скользишь, сравни.
- [ ] Упади до **≤35% здоровья** в бою → вспыхивают **Сила + Сопротивление** (Последний бой). Подлечись → пропадает.
- [ ] Никаких врождённых сопротивлений — огонь/тьма/эффекты бьют в полную силу.

### 🌑 Орк / 🪓 Гоблин / 🗡️ Снага
- [ ] Постоянное **Ночное зрение**.
- [ ] **Ночью или под землёй** → иконка **Сила** (Дети Тьмы).
- [ ] Съешь **гнилую плоть** → **нет эффекта Голод** (лужёный желудок).
- [ ] Выйди на **открытый дневной свет** → **Слабость (+Замедление у Орка/Снаги)** (Страх Солнца). В тень/ночь → пропадает.
- [ ] Гоблин: хорошо лазает по стенам; Снага: очень быстр и скрытен; все хилые.

### 🛡️ Урук-хай
- [ ] +3 сердца, самый сильный удар (+20%), но **медленнее скорость атаки** (−15%).
- [ ] Выйди на **открытый дневной свет** → иконка **Сила** (Презрение к Солнцу — наоборот, чем у прочих орков).
- [ ] Лужёный желудок (гнилая плоть безопасна); стойкость к голоду; **нет** Страха Солнца.

## 3. Опыт, древо и выборы
- [ ] Занимайся расовыми делами и смотри рост опыта: копай (mining), бей мобов (combat), стреляй (archery), крадись (stealth), используй **наковальню/стол зачар./кафедру** (runecraft), **нотный блок/проигрыватель** (song), взаимодействуй/катайся на **животных** (beast_lore), убей **чемпиона** / стой рядом с союзниками (leadership), убей **жителя** (shadow — орки), крафти (smithing), ешь/новый биом (survival). Проверяй `/kindreds inspect`.
- [ ] Набей очки: `/kindreds grantxp <дисциплина> 8000`. Открой **K**, кликни узел → панель справа покажет эффект/цену/требования → **Learn**. *Ожидается:* звон + золотая вспышка на узле.
- [ ] Открой ветку до **специализации**, затем посмотри на её соперника → он **ЗАКРЫТ** (ты выбрал один путь). Клик по нему: «другой путь закрыт».
- [ ] Дойди до **Великого капстоуна** (нужны узлы из нескольких дисциплин — показано «Requires …»).
- [ ] `/kindreds respec` → все узлы возвращены, эффекты сняты.

## 4. Конфиг (админ/сервер)
- [ ] `/kindreds config` — список всего.
- [ ] `/kindreds config enableBirthTraits false` → врождённые черты снимаются за пару секунд. Верни `true` → возвращаются.
- [ ] `/kindreds config xpRateGlobal 3.0` → опыт втрое. `/kindreds config deathPenalty HARDCORE` и т.д.

---

## Как выглядит «хорошо»
Каждый эффект должен быть **заметным** и **соответствовать расе** (Гном — толстый/медленный, Орк — царит ночью и слабеет днём, Эльф — быстрый/зоркий/хрупкий во тьме). Интерфейс — **чёткий, читаемый, без вылетов и фиолетовых/битых текстур**. Сообщай обо всём, что кажется пустым, неверным или багованным.
